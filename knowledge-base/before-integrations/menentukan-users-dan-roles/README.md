# Menentukan Users dan Roles

Pada bagian Settings - Users & Roles, anda dapat membuat Roles yang ditujukan untuk menentukan fitur-fitur mana saja yang bisa diakses oleh Users sesuai dengan kebutuhan anda.&#x20;

![Klik Settings - Users & Roles](<../../../.gitbook/assets/1\_useroles (1).jpg>)

![Klik Roles](../../../.gitbook/assets/2\_useroles.jpg)

![Klik Add Roles](../../../.gitbook/assets/3\_useroles.jpg)

Untuk roles yang sudah dibuat, anda juga bisa melakukan edit untuk mengubah akses terhadap fitur yang di-assign ke users.
